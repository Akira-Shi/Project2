Conditional Statements (if/else)

Conditional statements allow you to make decisions in your code.

if condition:
    # code block
else:
    # code block

Loops (for/while)

Loops help you run a block of code repeatedly based on a condition.

for i in range(5):
    print(i)

while condition:
    # code block
